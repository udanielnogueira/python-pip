# python-pip

Primeiros passos para usar o PIP.

## Instalando um pacote

``python -m pip install -U package_name``

### Exemplo

python -m pip install emoji

python -m pip install PySimpleGUI

## Atualizando o PIP

``python -m pip install --upgrade pip``
